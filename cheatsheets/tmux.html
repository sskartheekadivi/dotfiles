tmux

Keyboard reference

tmux 3.2a Kanagawa Dragon TPM v3.0.0 Prefix: C-a
C-a  Prefix — Ctrl + a
M-x  Meta — Alt + x  ·  no prefix needed
prefix x  press prefix first, then x

Sessions

KeyAction
M-↑Previous session
M-↓Next session
prefix $Rename session
prefix sInteractive session list
prefix dDetach

Windows

KeyAction
M-←Previous window
M-→Next window
M-cNew window (inherits CWD)
prefix ,Rename window
prefix wInteractive window list

Panes

Navigate

M-hFocus left
M-jFocus down
M-kFocus up
M-lFocus right

Resize

M-HResize left 5
M-JResize down 5
M-KResize up 5
M-LResize right 5

Create & Destroy

M-vSplit vertical — side by side (CWD)
M-sSplit horizontal — top/bottom (CWD)
M-zToggle zoom / fullscreen
M-xKill pane (prompts for confirmation)

Pane Join & Break

KeyAction
M--Pull pane in — horizontal split
M-=Pull pane in — vertical split
M-bBreak to new window (restores name)
M-BBreak to chosen session (tree picker)
Join & Break open an interactive tree picker — use arrow keys to select the target.

Copy Mode

vi keys · selection copied to system clipboard via xclip / wl-copy
KeyAction
[Enter copy mode
vBegin selection
C-vToggle rectangle (block) selection
yYank to clipboard & exit
EscapeCancel / exit
Hold Shift and drag to use the terminal's native selection, bypassing copy mode.

Prefix Commands

KeyAction
prefix C-aSend literal Ctrl+a to terminal
prefix rReload config
prefix :tmux command prompt
prefix ?List all key bindings

Plugin Manager — TPM

KeyAction
prefix IInstall plugins
prefix UUpdate plugins
prefix M-uRemove unused plugins

Popup Shell

KeyAction
M-pFloating shell in current pane's directory
Press C-d or type exit to dismiss.

Mouse

Mouse is enabled — click to focus a pane or window, drag to scroll.

Hold Shift to bypass tmux and use native terminal selection.
